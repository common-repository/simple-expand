Easy and lightweight read more functionality, wrap the text inside ´[expand]´ tags where ever you want.

== Description ==
Usage of the expand shortcode:
[expand title="My title"]
The content goes here.
<img src="even_images_can_go_in_here" />
And wrap up with a closing expand tag.
[/expand]

== Installation ==
1. Upload "/simple-expand/" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Place [expand] shortcodes on your page.
